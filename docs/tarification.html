Options de Tarification

Nous proposons plusieurs forfaits pour répondre à vos besoins d'apprentissage. Choisissez celui qui vous convient le mieux :

  • Forfait Mensuel

    Accès illimité à tous les cours pour une durée d'un mois.

    Prix : 29,99 € par mois

  • Forfait Annuel

    Accès illimité à tous les cours pendant un an, avec une remise de 20 %.

    Prix : 239,99 € par an

  • Forfait Étudiant

    Accès à tous les cours à un tarif réduit pour les étudiants (justificatif requis).

    Prix : 19,99 € par mois

Oui, vous pouvez annuler votre abonnement à tout moment depuis votre profil utilisateur.
